## 3. Main System Flow Chart

```
START
  |
  ├-► Power On System Initialization
  |   ├-► Hardware Check
  |   ├-► Load System Configuration
  |   └-► Connect to Charging Dock
  |
  ├-► Environment Scanning
  |   ├-► Activate Sensors
  |   ├-► Map Surroundings
  |   ├-► Identify Electronic Devices
  |   └-► Detect Human Presence
  |
  ├-► Navigation Mode
  |   ├-► Receive Task Assignment
  |   ├-► Calculate Optimal Path
  |   ├-► Move to Target Location
  |   └-► Execute Task
  |       ├-► Control Devices
  |       ├-► Health Scan
  |       └-► Collect Data
  |
  ├-► Data Processing
  |   ├-► Analyze Sensor Data
  |   ├-► Update Device Status
  |   ├-► Process Health Data
  |   └-► Generate Reports
  |
  ├-► Communication
  |   ├-► Sync with Charging Dock
  |   ├-► Update User Interface
  |   └-► Send Alerts if Required
  |
  ├-► Battery Monitoring
  |   ├-► Check Battery Level
  |   ├-► If Battery < 20%
  |   |   └-► Return to Dock
  |   └-► Continue Operation
  |
  └-► Repeat Main Loop
```
